## Version 1.0 (tests) Release Notes

Cette version est dédiée aux premiers retours utilisateurs, les données utilisées sont issues du dernier import de l'extraction INPN. Elle comprend :

- Les premiers filtres généraux sur :
  - Groupes taxonomiques
  - Taxons
  - Limites temporelles
  - Limites géographiques
    - Départements
    - Communes
- Un composant Filtres Avancés que l'on peut utiliser en fenêtre flottante ou dans la barre latérale
- Les couches de restitution suivantes:
  - Commune
  - Grille 5x5 nationale
  - Grille 10x10 nationale
- Quelques exemples de couches thématiques dédiées à l'environnement : cette partie fera l'objet d'une refonte complète via l'intégration d'une fonctionnalité dédiée
- Quelques couches de limites administratives
- Une fenêtre d'accueil : le contenu final n'étant pas statuer, les textes sont remplacés temporairement par du lorem ipsum
- Un tutoriel de première connexion
- Un premier lien vers les métadonnées des cadres acquisition et jeux de données
- Une première version des fiches de métadonnées cadres acquisition et jeux de données (sans les liens vers les flux wms / wfs)

Elle ne comprend pas :

- Le design final de l'application et sa compatibilité avec les tablettes et smartphones (responsive design) même si quelques adapatations ont déjà été implémentée
- L'intégralité des données producteurs
- La compatibilité avec la recherche générique de mviewer
- D'options de filtrage avancés
